# Cartão de Crédito App

## Descrição
Este projeto é uma aplicação simples em C# que identifica a **bandeira de um cartão de crédito** (Visa, MasterCard, etc.) com base no número informado.  
O objetivo é praticar conceitos de programação, lógica e utilização do **GitHub Copilot** para acelerar o desenvolvimento.

---

## Funcionalidades
- Recebe o número do cartão de crédito do usuário;
- Identifica a bandeira do cartão (Visa, MasterCard, American Express);
- Retorna a bandeira correspondente no console;
- Aplicação feita em **um único arquivo C#**, sem dependências externas.
